# War Library — Comprehensive Codebase Audit

**Date**: March 23, 2026
**Auditor**: Claude Opus 4.6 (automated, 8 parallel agents)
**Scope**: Full codebase, data integrity, security, CI/CD, GitHub repo, humanitarian fact-check
**Status**: Report only — no fixes applied

## Executive Summary

**Total findings: 150** across 8 audit domains

| Severity | Count | Key Themes |
|----------|-------|------------|
| Critical | 17 | Data corruption risk, duplicate inflation (~2,055 false fatalities), prompt injection, missing auth middleware |
| High | 28 | Unauthenticated endpoints, CSRF bypass, stale humanitarian figures, CI failures ignored, no branch protection |
| Medium | 58 | Performance bottlenecks, race conditions, hydration mismatches, schema inconsistency, missing tests |
| Low | 47 | Accessibility gaps, dead code, minor config, cosmetic data issues |

**Prior fixes confirmed holding**: Security headers, timing-safe auth, flock guard, JSDOM memory leak, sitemap cleanup, RSS feed fixes, About page analytics disclosure — none re-flagged.

**Core unresolved problem**: Pipeline deduplication is structurally insufficient. The same event described by different sources produces different text, bypassing word-overlap matching. Minab school bombing was cleaned up previously but has 11 new duplicates. IRIS Dena is triple-counted again. Three new cumulative death tolls slipped through. **Estimated false fatality inflation: ~2,055 out of ~3,627 in events_latest.json.**
